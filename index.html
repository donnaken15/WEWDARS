or Wesley's Extreme Windows Denormifying and Anticorpo Ricing System

Super optimize your installation of Windows by running this singular batch script, and
have an interface that looks just like mine! 
Required to run on, and only tested on, Windows 10 or AtlasOS.


Simply enter these two commands in the command prompt:
curl -LO https://donnaken15.github.io/WEWDARS/main.bat
main.bat█

If you don't have curl, which most
Windows 10 machines should have, download it here.
This can be put alongside the batch script
or in the Windows or system32 folder.

NOTE, that this is designed for power users and super users (like me),
and is recommended to be used on a computer that solely belongs to you,
or has only one user profile available on it.

It's also recommended you do this on a fresh install or with no files present
that could be effected (even when humanely checked to be safe) if you plan
to run TronScript.
The deets:
This script culminates of a bunch of leet hax and tools
to make the Windows experience more sophisticated to
use, and get down to the bedrock of the system.
Whether you want maximized performance and a minimal process count,
or a development environment or workflow similar to Linux but still
working with Windows applications, or both, by running this tool,
you'll get to make Windows yours!
